# Flipkart Clone

A static Flipkart-style shopping UI built with HTML, CSS, and vanilla JavaScript.

## Project Structure

- `index.html` - root entry for static hosts
- `filpkart.html` - main homepage
- `style.css` - shared styling
- `script.js` - cart, filters, wishlist, and carousel behavior
- `mobile.html`, `laptop.html`, `fashion.html`, `f.html`, `Electronincs.html`, `game.html` - category pages
- `vercel.json` - Vercel routing configuration

## Deploy on Vercel

1. Push this repository to GitHub.
2. Import the repository in Vercel.
3. Use these settings:
   - Framework Preset: Other
   - Build Command: leave empty
   - Output Directory: leave empty
4. Deploy.

The root route `/` is configured to open `filpkart.html`.

## Run Locally

Open `index.html` or `filpkart.html` in a browser.
